# DSM_AutoChangeWallpaper
本项目仅适用于群晖 DSM 7.2.x 及以上系统。
包含"AutoChangeWallpaper.sh"脚本文件。<br>
支持 使用 本地图片 或 必应每日一图 做图源。<br>
支持 用户壁纸 和 群晖 登录壁纸分开配置。<br>
支持 收集保存必应每日一图到指定路径。<br>
